# 🎯 Jogo de Adivinhação (Python)

Projeto simples de jogo de adivinhação desenvolvido em Python, onde o jogador tenta descobrir um número aleatório gerado pelo sistema.

## 📌 Sobre o projeto
Este projeto foi criado com o objetivo de praticar lógica de programação, estruturas de repetição, validação de entrada e uso de números aleatórios.

## 🚀 Funcionalidades
- 🎲 Geração de número aleatório
- 📊 Sistema de níveis de dificuldade (0 até 10 bilhões)
- 🔁 Loop contínuo até o usuário acertar
- 📈 Contador de tentativas
- ⚠️ Validação de entrada do usuário
- 🔄 Opção de jogar novamente

## 🛠 Tecnologias utilizadas
- Python

## ▶️ Como executar

1. Clone o repositório:
git clone https://github.com/SanttsFellype/jogo-adivinhacao-python.git  

2. Acesse a pasta:
cd jogo-adivinhacao-python

3. Execute o arquivo:
python jogo_adivinhacao.py

## 📷 Exemplo de uso

Qual dificuldade você quer jogar?  
1- 0 a 10  
2- 0 a 100  

Você acha que o valor sorteado é quanto?  
50  

50 é maior do que o número sorteado, tente novamente!
